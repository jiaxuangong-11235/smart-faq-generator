FAQs for Generative AI project ideas

Q1. What is the maximum team size for the generative AI projects?

Teams can consist of 1 to 3 members.

Q2. What are the required deliverables for the projects?

Each project must include a documented GitHub repository, a short report, and a short demo video showing the project in operation.

Q3. When is the full project report due?

The full project report is due by midnight on December 1.

Q4. What is the purpose of the mid-project check-in?

The mid-project check-in, due on November 15, requires teams to provide updates on their progress and any significant changes to their project idea.

Q5. Are there specific tools recommended for the projects?

Yes, suggested tools include Python, Hugging Face Transformers, Google Colab, and various LLM APIs like OpenAI and Anthropic.

Q6. What should teams do if they pivot strongly from their original project idea?

If a team pivots strongly from their original idea, they should discuss it with the instructors before the mid-project check-in date.

Q7. Is there a requirement for a website for the projects?

Yes, for 2- and 3-person teams, a working website showcasing the project is required.
